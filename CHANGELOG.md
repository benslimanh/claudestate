# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-15

### Added

- Initial Python CLI for `STATE.md` project memory management
- Commands: `init`, `sync`, `status`, `checkpoint`, `blocker`, `context`, `loop-check`, `arch-update`
- `CLAUDE.md` sync support for Claude Code startup context
- Companion tools for Go and Rust integrations
- GitHub Actions CI for linting, type checks, and tests
